💡 바코드 생성기 사용 가이드

📱 기본 사용법

  • 처음 실행 시 직번을 입력하면 자동으로 바코드가 생성됩니다
  • 저장된 직번은 앱을 다시 실행해도 유지됩니다
  • 직번은 숫자만 입력 가능합니다 (최대 7자리)
  • 우측 하단의 ✏️ 버튼으로 직번을 수정할 수 있습니다

⚡ 편리한 기능

  • 홈 화면에 추가하여 앱처럼 사용할 수 있습니다
  • 다크모드를 지원합니다 (시스템 설정 연동 가능)
  • 좌측 하단에서 테마를 변경할 수 있습니다
  • 설정에서 바코드 크기를 조절할 수 있습니다

⚠️ 주의사항

  • 카카오톡 인앱 브라우저에서는 일부 기능이 제한됩니다
  • 원활한 사용을 위해 기본 브라우저 사용을 권장합니다
  • 브라우저 데이터 삭제 시 저장된 직번이 삭제될 수 있습니다
  • Code-128 형식으로만 바코드가 생성됩니다

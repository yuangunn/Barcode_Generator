사용 가이드
바코드 생성
  • 프로필을 추가하면 자동으로 바코드가 생성됩니다
  • Code-128, EAN-13, Code-39, QR 코드를 지원합니다
  • 여러 프로필을 저장하고 탭으로 전환할 수 있습니다
편리한 기능
  • 홈 화면에 추가하면 앱처럼 사용 가능합니다
  • 오프라인에서도 바코드 생성이 가능합니다
  • 다크모드 지원 (시스템 설정 연동)
주의사항
  • 카카오톡 인앱 브라우저에서는 일부 기능이 제한됩니다
  • 브라우저 데이터 삭제 시 저장된 프로필이 삭제됩니다
  • EAN-13은 정확히 12~13자리 숫자가 필요합니다
